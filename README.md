Este proyecto tiene un valor muy especial para nosotros porque fue nuestro primer "gran proyecto" en la universidad. Fue el resultado de nuestra primera Integradora, donde tuvimos el reto de dejar de hacer prácticas sueltas y conectar tres mundos en un solo sistema: Web, Móvil y Base de Datos.

## ¿De qué trata?

Luxor es una tienda en línea completa (tipo Amazon o Mercado Libre) que desarrollamos nativamente para Android.
Lo más interesante es la **sincronización total**: la app "habla" en tiempo real con la versión Web. Si alguien compra un producto en la computadora, desaparece al instante del inventario en el celular. Ambas plataformas sirven para lo mismo: comprar y vender artículos.

## Lo que hace la App

* **Compra y Venta:** Cualquiera puede ser vendedor, subir sus fotos y publicar productos, o solo navegar para comprar.
* **Carrito Inteligente:** Puedes agregar cosas de diferentes vendedores, ver sumas totales y administrar tu lista antes de pagar.
* **Simulación de Pagos:** Hicimos una pasarela de pagos simulada que valida tarjetas y aprueba o rechaza la compra.
* **Rastreo de Pedidos:** Después de comprar, puedes ver si tu paquete está "En preparación" o "Entregado" desde tu perfil.
* **Cuentas Seguras:** Login y registro para proteger los datos de todos.

## Tecnologías

* **Kotlin:** Para que la app vuele en Android.
* **MySQL:** Donde guardamos todos los datos de usuarios y ventas.
* **API REST:** El puente que construimos para conectar el celular con la base de datos.
* **MVVM:** La estructura que usamos para tener el código ordenado.
